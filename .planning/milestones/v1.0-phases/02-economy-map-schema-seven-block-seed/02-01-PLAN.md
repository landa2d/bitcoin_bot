| browser anon → PostgREST → `economy_map.*` | The browser anon key authenticates to PostgREST; RLS policies + Accept-Profile header gate which rows/tables anon sees. This boundary is NEWLY created by this phase (previously, the browser only saw `public.newsletters` + `public.spotlight_history`). |
