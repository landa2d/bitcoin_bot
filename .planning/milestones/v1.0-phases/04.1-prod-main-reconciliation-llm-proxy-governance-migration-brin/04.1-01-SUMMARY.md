## Decisions Made
- **Fail-loud event_type = `'cap_missing'`** (Plan-recommended; clearer audit trail than overloading `'cap_hit'`). The exact string is named in a SQL comment in migration 034 SECTION 5 and matches the two `_emit_governance_event(..., "cap_missing", ...)` call sites in proxy.py (unknown-agent branch and missing-cap branch). The downgrade path reuses the already-whitelisted `'model_downgrade'` type, so no extra type was needed for it.
- **Downgrade reservation reconciliation = full-refund-then-re-reserve.** Governance runs after `reserve_balance`, so the reservation sits at the old model's `estimated_sats`. The downgrade branch settles the old reservation at 0 (full refund) and calls `reserve_balance` again at the new model's `ESTIMATED_COST_SATS`, rebinding `balance`/`estimated_sats`. Chosen over settle-the-difference because the reservations are tiny (single-digit/low-double-digit sats) and refund+re-reserve is unambiguously correct (handles the deepseek 2 vs sonnet 80 sat delta, and re-checks balance sufficiency at the new model). A code comment documents it at the chat call site.
- **Cross-provider downgrade decision = typed redirect (option ii), not body translation.** gato's only downgrade is `claude-sonnet-4-20250514` (anthropic, `/v1/messages`, anthropic body) → `deepseek-chat` (deepseek, `/chat/completions`, OpenAI body). The anthropic handler does NOT rebind `route` and POST an anthropic body to a deepseek URL. Instead it refunds the reservation and returns HTTP 429 `{"type":"governance_downgrade","new_model":"deepseek-chat","retry_endpoint":"/v1/chat/completions"}`; `check_governance` has already emitted the `model_downgrade` event so it is auditable. A defensive same-provider branch (future anthropic→anthropic target) rebinds inline. The chat handler does inline translate-and-dispatch because deepseek/openai share the OpenAI-compatible chat shape. The canary (Plan 02 §3) asserts the gato sonnet-over-cap call returns the typed redirect + the `model_downgrade` row — explicitly NOT a 200 sonnet pass-through and NOT a plain reject.
