|----------|-------------|
| edition_eval key (env) → LLM proxy | The governed identity secret; must reach only the anthropic client, never a log/DB row/flags object |
| draft/verdict data → operator alert + edition_evals rows | Untrusted prose used to build reason/alert text (log-injection surface) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-30-KEY | Information disclosure | LLM_PROXY_EVAL_KEY | mitigate | `_build_eval_llm_client` reads the key via `_get_eval_api_key()` and passes it only to `anthropic.Anthropic(api_key=...)`; logs a boolean, never the key; the key never lands in a DB row or the flags object (GOV-01) |
