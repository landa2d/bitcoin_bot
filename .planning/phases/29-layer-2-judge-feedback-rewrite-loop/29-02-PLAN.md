<objective>
Build the feedback-rewrite loop on top of Plan 01's judge + scoring: the targeted revise call (`_revise_draft`), the structured feedback builder (`_build_feedback`), the best-attempt selector (`_select_best_attempt`, D-11), and the full N=2 loop in `run_layer2` that replaces the Plan-01 attempt-0 extension point. When a judge dimension fails, the loop re-calls the writer with a targeted "fix exactly these issues, invent nothing" revise (both bodies as a unit, D-08), re-judges, and terminates hard at N=2 rewrites — returning `passed` when the rewrite fixes it, or `held_voice` with the best attempt when it does not.

Purpose: This is the heart of Layer 2 — the bounded rewrite loop. The revise is a targeted `_llm_call` (writer-agnostic, one function serves single_pass and block_v1) with the source-facts guardrail, NOT a full writer re-run (D-07). N=2 is a hard cap with no best-effort publish (LOOP-02); a continuity hard-fail still gets its rewrite chance (severity ≠ rewrite-eligibility, D-06); and `held_voice` returns the BEST attempt, not the latest, consuming the per-attempt scoring the telemetry produces (D-11). The rewrite SAFETY re-check (re-running the Layer-1 gate per rewrite, D-01/D-02/D-03) and the telemetry finalization are Plan 03.

Output: `_revise_draft` + `_build_feedback` + `_select_best_attempt` + the completed N=2 loop in `run_layer2`, plus the LOOP-01/LOOP-02 + D-06/D-08/D-11 tests. Build-only, worktree-safe — no supabase, no `edition_evals` write, no live proxy, no container rebuild (D-09).
</objective>
