## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Rewrite draft prose → re-run Layer-1 gate | The rewrite is untrusted LLM output; it is fully re-verified before acceptance, and a new fabrication hard-aborts (D-01/D-02) |
| injected http_client → _CachingHTTPClient → gate network layer | The caching wrapper forwards only to the gate's SSRF-guarded fetches; the module adds NO raw egress of its own |
| attempts telemetry → Phase 30 write_eval_row | The module's returned telemetry is the persistence contract; it must fit write_eval_row params + verdict-iff-ok, but the module itself writes nothing |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-29-FABRW | Tampering | A rewrite hallucinates a new entity/stat under specificity pressure | mitigate | Full Layer-1 re-run on EVERY rewrite via `run_deterministic_gate` → `held_fabrication`, keep the clean attempt-0 draft; the fabricated rewrite is never returned (D-01/D-02) |
| T-29-SSRF | Info disclosure / SSRF | Prompt-injected draft prose triggers an outbound fetch to an internal host | mitigate | The re-check reuses the gate's inherited `_is_safe_public_url` guard (deterministic_gate.py:411); `_CachingHTTPClient` only forwards to the gate's already-guarded fetches — the module adds NO new network egress |
| T-29-ZERO | Tampering | A cached transient error masking a real fabrication, or a silent zero | mitigate | Only `fabrication` outcomes are consequential and stable under caching (404 never retried, 200 stays 200); a cached transient 5xx affects only `unverified`, which never holds/aborts (D-03); a judge error → `escalated`, never a fabricated 0 |
| T-29-DOS | Denial of Service | Runaway rewrite loop + network re-checks burn budget | mitigate | N=2 hard cap (LOOP-02) bounds re-checks to ≤2; the dedup cache prevents re-fetching unchanged refs (D-01); the governed wallet `on_cap_behavior='reject'` (GOV-02) is the backstop |
| T-29-LOG | Tampering (log injection) | Logging raw rewrite/flag prose at INFO | mitigate | Log only attempt numbers, verdicts, flag kinds/counts — never raw body/flag prose at INFO |
| T-29-SC | Tampering | pip installs | accept | Zero new packages this phase (stdlib + in-repo engines already present); no install task exists |
